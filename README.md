# 💰 Smartfinance

A full-stack personal finance management app with smart savings insights and data-driven recommendations.

## 🔗 Live Links

- **Live App:** https://smartfinance-n68m.vercel.app
- **Backend API:** https://smartfinance-fhub.onrender.com

## ✨ Features

- 🔐 Secure user authentication (JWT)
- 💸 Income/expense transaction tracking
- 📊 Visual analytics (pie & line charts)
- 🎯 Salary, savings goal & emergency fund planning
- 🧠 Smart financial insights and recommendations
- 📈 Spending category breakdown

## 🛠️ Tech Stack

**Frontend:** React, React Router, Chart.js
**Backend:** Node.js, Express
**Database:** PostgreSQL (Neon), Sequelize ORM
**Deployment:** Vercel (frontend), Render (backend)

## 📦 Run Locally

### Backend
\`\`\`bash
cd server
npm install
node index.js
\`\`\`

### Frontend
\`\`\`bash
cd client
npm install
npm start
\`\`\`
